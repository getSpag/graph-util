# Graph-util
### cool tool for edge-weighted directed graphs



![node4](https://github.com/user-attachments/assets/5bdbf29c-9513-4ab7-9775-7e998e56a32c)



**run in your browser, instructions on the page:**
* All clicks are sticky. 
* Left click any element to drag (drag edges by the dot). 
* With an element being dragged, left-click on 'Del' to delete. Anywhere else to place. 
* 'Add' adds a node. 
* Right click a node to begin an edge. Right click on another to finalize edge. Weight not yet mutable.
* In the future this will have DFS/BFS buttons to visualize each algo on the graph you've made.
